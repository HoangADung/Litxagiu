INSTALL AND PLOT GRAPH USING GNUPLOT

0)Put TichChap.cpp; TichChap.txt in the same folder before running code. Ex: C:\Users\Admin\Downloads\Litxagiu. Run code

1)Download and setup Gnuplot: Default path C:\Program Files\gnuplot 
https://sourceforge.net/projects/gnuplot/

2)Setup environment: Control Panel -> System and Security -> System -> Advanced system settings -> Environment Variables (Click System Variable/Path) -> Edit -> New

3)Paste "C:\Program Files\gnuplot\bin" 

4)OK, close, accept all

5)Open Command Prompt

6)Type "cd> C:\Users\Admin\Downloads\Litxagiu" , or any place that you put all files 

7)Enter

8)Type "gnuplot"

9)Enter

10)Type "plot 'Litxagiu.txt' w l"
